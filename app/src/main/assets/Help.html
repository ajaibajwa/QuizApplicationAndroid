Login Screnn

  • This Screen open automatically on launching the app,if no user has logged in. Firstly,the User has to Create an account from the register screen by clicking on "NEW USER?" Button at the bottom of Screen

Login

  • Next screen is a Login screen in which user can login to their account if they have an account.
  • Login screen shows email and password option to get the information from user.
  • "Remember me!" option is also available on this screen which would save the login data of user and fills the appropriate fields on next login.
  • Another option is a register button on this screen.
  • This option is for all the users which are not registered yet.

Registration

  • In this screen user have to give all the required information regarding them.
  • Like name,email address, password, address, date of birth ,car plate number and many more.
  • After registration user will go to login screen and to access the application.

Menu Items

  • After successfully login the menu item screen should be displayed on the screen.
  • In this screen, all the items are shown which are included in application and the user is able to use these items.
  • These items are further linked to its functionality page.

Parking Payment Receipt

  • In this screen, accept different values from the user in a systematic manner.
  • After accepting a value, all the values are displayed on the next screen in a prper manner

Parking Location

  • In this screen, the adject current location of the parking is displayed.

Parking Report

  • This screen shows the information about all the parking done so far.

User Profile

  • This screen allow the user to update and save the whole data about him/her.

Parking Manual

  • In this screen application tells the user that how to use the application, that is, guideline for the user.

Support Contact

  • In contact support screen, user can use the different applications like call, message, email from contact list.

Logout

  • After clicking the logout button, user would automatically go back to the login page.
